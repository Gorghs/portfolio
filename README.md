# Karthick Portfolio

Professional portfolio showcasing AI systems, automation engineering, and applied backend workflows.

## Overview
- Focus: AI workflow automation, semantic systems, and practical engineering delivery.
- Built with React, TypeScript, Vite, and GSAP.

## Run Locally
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`

You are free to use, modify, and adapt this project for your own portfolio or learning. No attribution required.
